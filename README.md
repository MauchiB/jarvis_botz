# Jarvis Botz

Jarvis Botz is a modular and extendable Telegram bot built with Python.  
The project is currently under active development and will continue to evolve over time.

---

## Features (current & planned)
- Basic Telegram bot functionality
- Role system (`developer`, `admin`, `usr`)
- AI-powered responses (via different LLM providers)
- Modular architecture with handlers and pipelines
- SQLAlchemy-based database layer
- Planned integrations: Redis, PostgreSQL, additional LLMs

---

## Tech Stack
- Python
- python-telegram-bot
- SQLAlchemy
- LangChain
- Docker (optional)
